# Pense em Python 2026 – Repositório de Estudos

## Sobre este repositório

Este repositório reúne meu material de estudo do livro/curso “Pense em Python”, versão 2026.
Aqui vou concentrar exemplos, exercícios resolvidos, anotações e pequenos projetos desenvolvidos ao longo do aprendizado.

O objetivo é acompanhar minha evolução em Python de forma organizada e versionada, usando Git e GitHub.

## Objetivos do estudo

- Desenvolver uma base sólida em Python.
- Praticar lógica de programação e resolução de problemas.
- Ganhar familiaridade com Git e GitHub no dia a dia.
- Criar pequenos scripts e exercícios que possam servir de referência futura.
- Organizar o conteúdo por capítulos e temas para consulta rápida.

## Fluxo de uso com Git

Sempre que eu estudar ou fizer mudanças no código, a rotina será:

git status # Ver o que mudou
git add . # Adicionar arquivos modificados
git commit -m "descrição" # Registrar o que foi feito
git push # Enviar para o GitHub

## Exemplos de mensagens de commit:

- Adicionar exercícios do capítulo 2
- Resolver exercícios de funções do cap03
- Organizar arquivos em pastas por capítulo
- Anotações sobre listas e dicionários
