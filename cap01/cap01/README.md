# Capítulo 1 – O caminho do programa

Neste capítulo, comecei a aprender os conceitos básicos de programação em Python e a forma de pensar como programadora.

## O que aprendi

- Como Python lida com **números** e **operações matemáticas simples**.
- Como usar o Python como uma **calculadora** para resolver problemas do dia a dia.
- A diferença entre **programa**, **linguagem** e **interpretador**.
- A ideia de **algoritmo**: uma sequência de passos para resolver um problema.
- O que é um **script** e como salvar e executar um arquivo `.py`.
- Noções iniciais de **ambiente de desenvolvimento** (VS Code + venv).

## Exercícios resolvidos

- Converter **minutos e segundos em segundos** totais.
- Converter **quilômetros em milhas** usando uma taxa de conversão.
- Calcular o **passo médio** (tempo por milha) e a **velocidade média** (milhas por hora) de uma corrida.

## Objetivo do capítulo

Entender que programar é, principalmente, aprender a **expressar problemas em passos lógicos** e usar Python como ferramenta para automatizar esses cálculos.
